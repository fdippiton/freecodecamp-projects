Introduction

CSS are stylesheets used to give styles to websites.

CSS means Cascades Stylesheets.

Some css attributes are:

  • font-famiy
  • font-size
  • color
  • width
  • padding
Selectors

To style html elements on css, selectors are key.

There are diffents types of selector such as class selectors, id and elements.

- Class selector syntax

.className { ... }

- Id selector syntax

#idName {... }

- Element selector syntax

elementName {... }
Flexbox

Flex-box is a layout alternative which allows the correct distribution of the content on the websites.

Some flex-box atributes are:

justify-content

It takes values like:

flex-start, flex-end

align-items

It takes values like:

top, center, bottom

flex-direction

It takes values like:

column, row, column-reverse, row-reverse
Colors

On CSS it's possible to give color to elements and text.

There are three alternatives to do this:

  • Hue

  • RGB

  • Hexadecimal

Text Styles

To style the text it's precise to use the following atributes:

  • font-family

  • font-size

  • font-weight

  • line-height

  • color

Media Queries

Media queries are used to make websites responsive

- The media query syntax

@media (dimensions){...}
